1. Формат файла

Файл состоит из описаний "боевых" параметров для PC и всех классов NPC, существующих в игре. Описания отделяются друг от друга пустой строкой. Каждое описание состоит из названия класса (уникально) и характеристик. Ниже приведен список характеристик по их алфавитному названию. Обозначение string значит, что на этом месте должна быть строка, N - числовое значение, mixed - или строка, или значение.

[Имя Класса]

Этот параметр должен быть указан первым. Например, [Enclave Patrol], [Fighting Mantis] или [Arroyo Villager]. Класс игрока называется Player AI. При описании "своих" NPC, т.е. членов своей команды в игре (party NPC), для каждого из них должны указываться следующие классы:

[PARTY NAME AGGRESSIVE]
[PARTY NAME BERSERK]
[PARTY NAME COWARD]
[PARTY NAME CUSTOM]
[PARTY NAME DEFENSIVE]

, где NAME - это имя члена команды, например: DOC, LENNY, GORIS и т.д.
 

aggression=N
Скорее всего, это общий уровень агрессивности, который используется для того, чтобы начать бой. Либо же это изначальная вероятность попадания (в процентах) данного NPC в своего противника. (вероятность изменяется в зависимости от Armor Class-а противника, расстояния, освещенности и т.д.). Во всяком случае, у тех классов противников, которые известны как достаточно агрессивные и опасные, это значение велико.
area_attack_mode=string
Режим стрельбы очередями. Существующие значения:

no_pref
always
sometimes
be_sure
be_careful
be_absolutely_sure


attack_end=N
attack_start=N

Боевые крики при атаке. В начале атаки, случайным образом выбирается число между двумя указанными значениями. Затем из файла master.dat\\text\english\game\combatai.msg выбирается и показывается сообщение с соответствующим номером. Порядок нумерации сообщений четко не определен, существуют общие правила (см. описание для файла combatai.msg).


attack_who=string

Как слышится, так и пишется. Указание на то, кого NPC должен атаковать в первую очередь. Существующие значения:

whomever_attacking_me
strongest
weakest
whomever
closest


best_weapon=string

Тип предпочтения оружия при атаке. Этот параметр существует не у всех NPC. Имеющиеся значения:

no_pref
never         (Deathclaw)
random
unarmed
ranged_over_melee
melee_over_ranged
unarmed_over_thrown    (Spore Plant)

Кстати, обратное последнему значение thrown_over_unarmed почему-то ни у кого не указано, и непонятно, есть ли такое предпочтение вообще  ;)))

Предположительно, задает общий стиль поведения "тела" NPC (перемещения etc). Существующие значения (синим наклонным шрифтом указаны примеры из игры):

Primitive

called_freq=N

Число, определяющее вероятность появления криков в бою для данного класса. Существующие значения - от 0 (не кричит вообще) до 1000000! (это у насекомого Mantis, которое, как известно, не кричит вообще ;))) А вот у класса Child (дети любят покричать, когда им делают больно, хе-хе >:) called_freq=10000.


chance=N

Скорее всего, вероятность критического попадания (Critical Hit Chance) NPC в своего противника. Существующие значения - от 0 до 35.


chem_primary_desire=N

Предпочтения при употреблении наркотиков. Порядковые номера предметов, или -1, если нет предпочтения. Если нужно указать несколько видов наркотиков, то они перечисляются через запятую. (напр. 284,81,103)


chem_use=string

Использование NPC наркотиков средств в бою. Существующие значения:

clean
sometimes
stims_when_hurt_little
stims_when_hurt_lots
anytime
always


color=N

Цвет. Вот только чего? ;)))))) Впрочем, для всех классов значение остается неизменным - 58. Возможно, этот параметр задает цвет обводки NPC тогда, когда NPC заслонен. (т.е. жёлтый цвет).
disposition=mixed
Предположительно, определяет поведение NPC в начале атаки. Существующие значения:

-1
none
сoward
defensive
aggressive
berserk


distance=string

Расстояние между NPC и его противником, тактика ведения боя. Существующие значения:

random (указан только для Mobsters и Tough Khan)
charge
on_your_own
stay
stay_close
snipe


font=N

Шрифт, используемый для печати боевых выкриков. Существующее значение в US-версии игры - 101. В остальных версиях встречаются также значения 102, 103, 104.


general_type=string

Так, например, класс Brain Mole Rat имеет body_type=Berserk Person, а внешний вид (general_type) - The Brain. Или класс LoPan: body_type=Shi, general_type=Lo Pan.

hit_torso_end=N
hit_torso_start=N

Крики NPC при ранении в глаза, пах, голову, левую руку, левую ногу, правую руку, правую ногу и туловище соответственно. Более подробно - см. описание переменной  attack_end .


hurt_too_much=string

Название переменной говорит само за себя. Предположительно, если эта переменная присутствует, то она указывает условие, при исполнении которого NPC считается слишком сильно раненным и падает на землю (возможно, пропуская при этом ход).  При этом обычно пишется, что Условия могут перечисляться через запятую.

crippled
blind
crippled_arms


max_dist=N

Неизвестно. Судя по некоторым признакам, речь идет о наибольшем расстоянии, на которое возможна стрельба данного NPC из оружия. Против того, что эта переменная задает наибольшее количество ходов (AP), разрешенных для данного NPC, говорит то, что у Gun Turret, например, max_dist=10. А, как известно, турельные пушки ног не имеют ;)
min_hp=N
Неизвестно. Можно было бы предположить, что тут задается минимальное значение Hit Points для NPC (в процентах?), но... Например, у End_Boss, т.е. у Frank Horrigan'а, min_hp=0. И наоборот, у Addict - Wimpy min_hp=100. Средние же значения - 10..20. Скорее всего, когда указан 0, то этот параметр не принимается во внимание.


min_to_hit=N

Может быть, минимальное количество очков для нанесения повреждения данному NPC. Или это может быть минимальное количество очков (без учета модификаторов), на которое NPC может поранить своего противника. Значение может быть 0.


miss_end=N
miss_start=N

Крики NPC при промахе. Более подробно - см. описание переменной attack_end .


move_end=N
move_start=N

Крики NPC при передвижении с целью атаки (т.е. когда еще не добежал до врага, а уже покричать охота ;) Более подробно - см. описание переменной  attack_end .


outline_color=N

Опять цвет, опять неизвестно чего, судя по названию - цвет обводки. Для всех классов NPC он равен 55.


packet_num=N

Предположительно, номер оружия, находящегося где-нибудь у NPC в загашнике. Слово Packet на военном жаргоне обозначает также пулю или снаряд - т.е. может быть, указывается номер боеприпаса. Наибольшее встречающееся значение - 186. Значение 0 встречается  только в описании класса самого игрока (класс Player AI).


run_away_mode=string

Вероятность побега NPC с поля боя. Существующие значения:

none
coward
finger_hurts
bleeding
not_feeling_good
tourniquet
never


run_end=N
run_start=N

Крики NPC при побеге с поля боя. Более подробно - см. описание переменной  attack_end .


secondary_freq=N

Неизвестно. Разброс значений очень велик - 5, 300, 200000.... Возможно, эта переменная является вспомогательной по отношению к called_freq.
